# spc-chart-engine

## Project setup

```bash
npm install

# npm install
# yarn install
```

### Compiles and hot-reloads for development

```bash
npm link spc-chart-engine
```

### Compiles and minifies for production

```bash
npm run build
```

## Usage

```bash
git clone https://github.com/JamesGrry/spc-chart-engine
cd spc-chart-engine
npm i
npm link spc-chart-engine
```
